# Монетизация FitFlow / GymMore

Документ с вариантами монетизации приложения: от простых MVP-способов до более зрелых B2C/B2B моделей.

## 1) Подписка (платные пользователи)

Самая предсказуемая модель по выручке.

### Вариант A: Freemium
- Бесплатно: базовые тренировки, трекинг, ограниченная аналитика.
- Premium: расширенная аналитика, история прогресса, продвинутые программы, персональные планы, приоритетные фичи.

### Вариант B: Tiered подписка
- `Basic` — минимум функций, низкий чек.
- `Pro` — полный доступ к аналитике, геймификации, социальным функциям.
- `Coach` — инструменты тренера (клиенты, шаблоны групп, отчеты).

### Что хорошо работает
- Пробный период 7–14 дней.
- Годовой план со скидкой 30–40%.
- Мягкие paywall-экраны на естественных точках ценности (после прогресса/рекорда).

## 2) Реклама

Подходит для бесплатного сегмента, но важно не ломать UX тренировки.

### Форматы
- Нативные блоки в ленте/контенте.
- Rewarded video за бонус (например, временный boost XP/доступ к премиум-отчету на 24ч).
- Баннеры только на второстепенных экранах (не в active workout).

### Ограничения
- Не показывать interstitial в критическом потоке: старт/лог тренировки/финиш.
- Ограничить частоту (frequency cap), чтобы не убивать retention.

## 3) Платные функции без подписки (a-la-carte)

- Разовая покупка готовых программ (сжигание жира, набор массы, домашние тренировки).
- Платные наборы шаблонов/планов от топ-тренеров.
- Платные визуальные темы/аватары/косметика (в геймификации).

Плюс: дополнительный доход даже от пользователей без подписки.

## 4) Тренеры и B2B

## B2C для тренеров
- Подписка тренера за доступ к кабинету и расширенным инструментам.
- Комиссия с платных программ тренера внутри платформы.

## B2B для залов/студий
- White-label или «клубный кабинет».
- SaaS-тариф по числу активных клиентов/тренеров.
- Корпоративные отчеты (посещаемость, вовлеченность, retention).

## 5) Партнерки и маркетплейс

- Партнерские офферы: спортпит, экипировка, умные весы/браслеты.
- CPA/CPS за покупки по реферальным ссылкам.
- Промокоды залов и локальных сервисов.

## 6) Монетизация геймификации

- Сезонный `Battle Pass` (миссии, косметические награды, эксклюзивные бейджи).
- Платные челленджи с призами и спонсорами.
- Командные соревнования (залы/клубы) с вступительным взносом.

Важно: не делать pay-to-win для ключевого прогресса здоровья.

## 7) Рекомендованная стратегия по этапам

### Этап 1 (быстрый старт)
- Freemium + Premium подписка.
- Минимальная нативная реклама для free.

Конкретный план запуска (для пользователей и тренеров):

1. Тарифы и доступы
   - `Free User`: базовые тренировки, базовый трекинг, ограниченная аналитика (последние 14 дней), 1 активная персональная цель.
   - `Premium User`: полная аналитика, безлимит истории, продвинутые цели, расширенная геймификация, отключение рекламы.
   - `Free Coach`: до 5 активных клиентов, базовый кабинет, ограниченные отчеты.
   - `Coach Pro`: безлимит клиентов, расширенные отчеты, шаблоны программ, приоритет в каталоге тренеров, без рекламы.

2. Цены и оффер (MVP)
   - `Premium User`: 299 ₽/мес или 2 490 ₽/год.
   - `Coach Pro`: 990 ₽/мес или 9 900 ₽/год.
   - Trial: 7 дней для обоих платных тарифов.
   - Скидка на годовой план: 30–35%.

3. Где ставим paywall
   - Пользователь: после просмотра расширенной аналитики, при попытке открыть историю >14 дней, при запуске персональных миссий.
   - Тренер: при попытке добавить 6-го клиента, открыть расширенный отчет, экспортировать план/результаты.
   - Ограничение: не показывать paywall во время активной тренировки/логирования подходов.

4. Реклама (только для free)
   - Формат: нативные карточки в ленте и контентных списках (не чаще 1 блока на 8–10 карточек).
   - Для `Free User`: реклама включена.
   - Для `Premium User`: реклама отключена.
   - Для `Free Coach`: реклама только в нерабочих экранах (например, лента/контент), не в рабочих экранах ведения клиентов.
   - Для `Coach Pro`: реклама отключена.

5. Ограничения по UX и бренду
   - Запрещены interstitial во время: тренировки, таймера отдыха, завершения тренировки, редактирования клиентского плана.
   - Частотный лимит: максимум 3 рекламных показа на пользователя в сутки.
   - Если CTR рекламы низкий и retention падает — сразу уменьшать частоту.

6. KPI на первые 30 дней
   - Конверсия в trial: `User >= 6%`, `Coach >= 8%`.
   - Конверсия trial -> paid: `User >= 35%`, `Coach >= 45%`.
   - Доля платящих: `User >= 2.5%`, `Coach >= 8%`.
   - Ad ARPDAU (free users): не ниже `0.8-1.2 ₽`.
   - Retention D30: падение не более 2 п.п. после включения рекламы.

### Этап 2
- Разовые покупки программ.
- Подписка тренера (Coach plan).

### Этап 3
- B2B для залов.
- Сезонная монетизация геймификации.

## 8) Метрики для проверки гипотез

- Конверсия в trial и в платную подписку.
- ARPU / ARPPU / LTV.
- Retention D1/D7/D30.
- Churn подписки (месячный/квартальный).
- eCPM и ad ARPDAU (если включена реклама).
- Доля выручки по каналам: подписка / реклама / разовые покупки / B2B.

## 9) Риски и анти-паттерны

- Слишком агрессивный paywall в core flow тренировки.
- Реклама в неподходящих местах (во время подходов/таймера отдыха).
- Перегрузка тарифов и непонятная ценность Premium.
- Платные фичи без ощутимой разницы результата.

## 10) Практический микс для FitFlow прямо сейчас

1. Запустить `Premium` для спортсменов: расширенный прогресс, история, персональные миссии.
2. Оставить free-режим с ограничениями по глубине аналитики.
3. Добавить осторожную нативную рекламу только в ленту/контентные экраны.
4. Для тренеров — отдельный `Coach` тариф.
5. Через 1–2 релиза протестировать разовые платные программы.
